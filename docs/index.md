# 综述·问卷情况

## 1.调研目的说明

本次调研目的是为了全面了解本校学生在备战研究生过程中的情况、需求和反馈，以便学校更好地研判学生的学习状态和生活需求，并提供有针对性的支持和服务。主要针对以下六大方面进行设计：了解学生的背景和决策过程，分析攻读研究生的动机，探讨专业型硕士的选择与优势，识别备考过程中的困难和需求，了解学生对教工和学校支持的看法，收集学生对学校服务和设施的反馈。

## 2.调研样本情况

- 收集时间：2023年11月03日开始
- 调研对象：本校九大学院中备战24考研的学生
